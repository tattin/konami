# konami.js

## what's konami command
  - press in following order
    1. up
    2. up
    3. down
    4. down
    5. left
    6. right
    7. left
    8. right
    9. b
    10. a

## usage
  - js
    ```
    <script src="konami.js"></script>
    <script>
    $("#hoge").konami({
    	"height": "100px"
    }, 500);
    </script>
    ```
## action
  - if user succeed in the konami-command input,
    then jQuery/animate() triggerd.
